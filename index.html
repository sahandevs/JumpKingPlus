Video tutorial

Text explaination

  1. Find a level you would like to play in the workshop section, download it using the Download button.
  2. Open the .zip file.
  3. Replace the Jump King/Content/mods folder with the mods folder inside the .zip file.
  4. (optional) Make sure the level.xnb and mod.xml file exist.
  5. Open Jump King and click on New Game (unless you have a save for it).
  6. Profit.

The quick restart button has been disabled by default from v1.5.2.

In order to enable it:

  1. Edit the JumpKing.exe.config file inside the Jump King folder.
  2. Find the current text in the file: boomerProofRestart="True".
  3. Change it to boomerProofRestart="True".

Video tutorial

Text explaination

  1. Find a skin/collection you would like to use in the workshop, download it using the Download icon button.
  2. Open the .zip file.
  3. Head to Jump King/Content/wardrobe.
  4. Drag the files from the .zip file inside the folder.

If some skins are in the way once you start the game, consider disabling them in the Inventory > Wardrobe.

Warning: with cheats, achievements are disabled!

Enabling cheats

Edit the JumpKing.exe.config file inside the Jump King folder; where it says cheats="False" change it to cheats="True".

Disabling cheats

Edit the JumpKing.exe.config file inside the Jump King folder; where it says cheats="True" change it to cheats="False".

JumpKingPlus has been approved on speedrun(dot)com for speedrunning. You will need to disable cheats (if you have them enabled, cheats enabled runs will be not valid). More details on the speedrun rules.
