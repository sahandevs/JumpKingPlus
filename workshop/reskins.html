How do I add a skin?

  1. Find a skin you would like to use in the workshop, download it using the Download button.
  2. Open the .zip file.
  3. Head to Jump King/Content/wardrobe.
  4. Drag the files from the .zip file inside the folder.
  5. Profit.

If a skin is in the way once you start the game, consider disabling it in the Inventory > Wardrobe.
